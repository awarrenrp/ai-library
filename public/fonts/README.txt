Basel Grotesk (optional)

Place licensed .woff2 files here so the “Basel Grotesk” preview can load them.

Expected names (edit src/basel-grotesk.css if your files differ):

  BaselGrotesk-Book.woff2      → font-weight 400
  BaselGrotesk-Medium.woff2    → font-weight 500
  BaselGrotesk-SemiBold.woff2  → font-weight 600

If these files are absent, the UI still uses the font-family stack and falls back to Inter.
